# COMP2005 Maternity Ward Management System

## Overview

This repository contains a Java-based maternity ward management system developed for the COMP2005 module at the University of Plymouth.

The project focuses on **Feature F2**, which allows users to search for patients who occupied a selected room within the previous seven days. The system combines backend filtering logic, a graphical user interface, and software testing practices.

---

## Features

* Search patients by room history within the last 7 days
* Simple Swing-based graphical user interface
* Backend processing using Spring Boot
* Input validation and error handling
* Automated unit and integration testing
* Continuous Integration using GitHub Actions

---

## Technologies Used

* Java
* Spring Boot
* Swing GUI
* JUnit 5
* Gradle
* GitHub Actions
* IntelliJ IDEA

---

## Running the Project

Run the backend:

```bash
./gradlew bootRun
```

Run tests:

```bash
./gradlew test
```

---

## Project Purpose

This project demonstrates software testing strategies, validation techniques, and quality assurance practices within a healthcare-related Java application.
